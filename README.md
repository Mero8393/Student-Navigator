# Учебный навигатор пары

Проект для группы ИСиП25-02П.

Приложение показывает:
- какая пара идёт сейчас;
- кабинет или место проведения;
- преподавателя;
- следующую пару;
- расписание на выбранный день;
- важные пометки: экзамен, практика, физкультура.

## Запуск

1. Скачать файл `Учебный_навигатор_пары_Финальная_версия.html`.
2. Открыть файл двойным щелчком в браузере.
3. Пользоваться приложением.

Интернет и установка дополнительных программ не нужны.

- `Учебный_навигатор_пары_Финальная_версия.html` — финальная версия.
